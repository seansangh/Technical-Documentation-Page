Introduction

JavaScript is a scripting language that is cross-platform.

  • Server-side JavaScript allows for the communication of an application with a web server to produce customized responses for the user.
  • Client-side JavaScript communication typically occurs via a browser (i.e. running scripts).
What You Should Know

It is expected that you already have an understanding of:

  • Using the internet.
  • General HTML knowledge.
  • Some experience with programming.
Javascript vs. Java

The main differences between Javascript and Java include the following:

  • Java has static typing and strong type checking.
  • Java has a more class-based object model while JavaScript has a more prototype-based model.
  • JavaScript is a more free-form language where you do not have declare all of the classes, methods, and variables used.

JavaScript Variables

JavaScript variables are used for values in your application. These variables are case sensitive (i.e. a-z and A-Z) and must start with either a letter, underscore symbol (_), or dollar symbol ($).

How To Declare Variables

In order to declare a variable, you can use the keyword var (var y = 500). If you don't use var (y = 500), then the variable may be declared as a global variable.

In addition to var, you can use the keyword let to declare a variable (let y = 500).

Scope of Variables

If you declare a variable in a function, then that variable will only be available within that function. If you declare a variable outside a function, then it will be a global variable available to the rest of your code.

Global Variables

In order to access global variables, you can use the window.variable syntax where the variable in question is placed after window.

JavaScript Constants

If you would like a variable to retain its value throughout, then use the keyword const (const PI = 3.145).

Types of Data

There are seven data types:

  • null (denotes a null value)
  • undefined (denotes a value that is undefined)
  • Number (0.01, 0.1, 1, 2, .......)
  • Boolean (true or false)
  • String ("word statements")
  • Symbol
  • Object

Declaring Functions

Functions are declared by the function keyword followed by the name of the function, a list of arguments to the function (seperated by a comma and enclosed in parentheses), and a statement defining the function between { }.

function myFunction(num) {

return num * num;

}
